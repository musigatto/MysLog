## MysLog
_MysLog_ es una aplicación gratuita diseñada para el seguimiento de tus entrenamientos. 
Con esta aplicación podrás:

- Crear y gestionar **sesiones** de entrenamiento.
- Seleccionar ejercicios entre un catálogo de más de **870** opciones, con filtros avanzados para facilitar la búsqueda.
- Consultar **instrucciones** detalladas e imágenes de cada ejercicio.
- Registrar ejercicios, **series** y **peso**.
- Visualizar **gráficas** y estadísticas para cada ejercicio.
- Guardar todo tu historial de entrenamiento de forma **local** y privada (100% _offline_).

Si bien toda la aplicación esta diseñada para funcionar en español e inglés, por favor, ten en cuenta que la base de datos se dispone (por el momento) únicamente en inglés.
